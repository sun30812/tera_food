# tera_food

기흥 테라타워 내 음식점을 랜덤으로 추천해주는 사이트 및 앱 입니다.

## 사용해보기

> 하기 링크에 접속하여 직접 사용해볼 수 있으며 가장 최근 버전(태그) 기준으로 업데이트 됩니다.

- 웹앱: [https://tera-food.web.app](https://tera-food.web.app)
- API 문서: [https://tera-food-doc.web.app](https://tera-food-doc.web.app)

## 참고

- 현재는 단순한 기능을 지원하나 나중에 더 많은 기능이 계속해서 추가될 예정입니다.
- 어느정도 기능 구현 시 Firebase Database에 음식점 정보를 저장하여 관리할 예정입니다.
- 현재는 일부 음식점 정보를 하드코딩하여 사용하고 있습니다.
